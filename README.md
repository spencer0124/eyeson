## Project Introduction

This is an iOS application developed to help visually impaired people enjoy art exhibitions. 
For individuals with total blindness, we recommend using this app in conjunction with the VoiceOver feature.

### Key Features
 1. Provides a list and description of artworks for each exhibition.
 2. Automatically links to the corresponding artwork description when a piece is photographed.
 3. Offers AI commentary (both objective and emotional).
 4. Allows for flexible artwork zoom-in.
 5. Enables users to share their art reviews.

## Development Environment

- Client : Swift, UIKit
- Backend : FastAPI
- Deployment : AWS EC2, S3, Github Actions
- Other Tools : Github, Notion, Swagger
